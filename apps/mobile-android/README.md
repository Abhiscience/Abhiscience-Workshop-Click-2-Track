# Workshop Click-2-Track - Android App

## Overview
Kotlin native Android app for vehicle movement tracking with mandatory photo capture at every stage.

## Features
- Role-based authentication
- Camera-first capture flow
- Emulator file-selection mode for testing
- Offline queue with Room database
- Plate recognition via backend API
- Job card matching and confirmation

## Build
```bash
./gradlew assembleDebug
```
